# Vue Budget

Applicazione web per la gestione del budget personale, sviluppata con **Vue.js**.  
Permette di tenere traccia delle entrate e delle uscite in modo semplice e intuitivo.

## Installazione

Clona la repository:

```bash
git clone https://github.com/RealComba/vue-budget.git
cd vue-budget
npm install
npm run dev
